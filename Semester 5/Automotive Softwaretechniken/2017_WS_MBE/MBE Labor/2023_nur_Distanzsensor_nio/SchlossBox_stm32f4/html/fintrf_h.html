5/*



6 * Detector defines MATLAB_TARGET_API_VERSION based on any combination



7 * (including none) of input macros of two kinds: legacy .vs. release-based,



8 * and hard .vs. soft.



9 *



10 * Legacy input macros (can be undefined or defined):



11 *     MEX_DOUBLE_HANDLE



12 *     MX_COMPAT_32



13 *



14 * Release-based hard input macros (can be undefined or defined to a legal release tag):



15 *     MATLAB_MEXCMD_RELEASE



16 *     MATLAB_MEXSRC_RELEASE



17 *



18 * Release-based soft input macro



19 *     MATLAB_DEFAULT_RELEASE



20 *



21 * Output macros:



22 *     MATLAB_TARGET_API_VERSION (must be defined)



23 *     2 legacy macros (could be defined or undefined)



24 *



25 * These are the generic rules:



26 * - Legacy macros are hard and serve as input and output macros.



27 * - Hard release-based macros cannot be used with legacy macros.



28 * - Soft macro is used only if there are no hard macros.



29 *



30 * Detector generates a preprocessor error if both legacy and hard released-based



31 * macros are defined.



32 *



33 * Detector defines release tokens to corresponding API version. In addition to



34 * the release macros Detector defines the following auxiliary macros:



35 *     MW_REL2VER	     â€“ Release-to-version macro



36 *     MW_FIRST_API_VERSION  â€“ First known serial version



37 *     MW_LATEST_API_VERSION - Version of the current release



38 *



39 * The first macro translates from a release token to a corresponding version.



40 * The latter two macros allow to generate a preprocessor error if a translated



41 * version is not between the first and the last versions.



42 *



43 * If none of the input macros are defined Detector sets MATLAB_TARGET_API_VERSION to



44 * MW_LATEST_API_VERSION or MW_FIRST_API_VERSION for SC ALTBUILD.



45 *



46 * If only the soft macro is defined Detector sets MATLAB_TARGET_API_VERSION to the



47 * version that corresponds to the proposed release token.



48 *



49 * Hard input macros override the soft macro.



50 */



51#define MW_FIRST_API_VERSION 700



52#define R2017a 700



53#define R2017b 800



54#define MW_LATEST_API_VERSION 800
